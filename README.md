# Memorial Digital

Sistema web para cadastro e visualização de memoriais digitais, permitindo homenagens, pesquisa de pessoas e gerenciamento dos registros.

## Tecnologias

- React
- React Router DOM
- Lucide React
- React QR Code
- CSS Modules

## Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/memorial-digital.git
```

### 2. Entre na pasta do projeto

```bash
cd memorial-digital-main
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Instale as bibliotecas utilizadas

```bash
npm install react-router-dom
npm install lucide-react
npm install react-qr-code
```

### 5. Execute o projeto

```bash
npm start
```

O projeto será iniciado em:

```
http://localhost:5000
```

## Funcionalidades

- Pesquisa de memoriais
- Filtro por categoria
- Página individual do memorial
- Galeria de imagens
- Geração de QR Code
- Painel administrativo
- Cadastro e edição de memoriais
